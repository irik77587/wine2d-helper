# Compile Direct2D apps using WINE GCC

I was following Direct2D tutorial from MS Docs and Made working Direct2D Demo.
Then I tried to compile it using winegcc but faced the problem. The tutorial 
taught me how to write assisted code. But winegcc compiles actual code.
No assisted code is supported. So with the power of AI, I demonstrate that you
don't need to change the source to write cross compatible code. I just need to
make your own code assist.
